# Koala-CommandLineTools

---

Some functions of the koala can be used from the command line.

## Description
This program has the following functions.
* Search book in Kandai
* Parchase request
* Check deadline

## Install
```
$ git clone 

$ cd Koala-CommandLineTools
$ python setup.py install
```
